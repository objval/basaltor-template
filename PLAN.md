# Basaltor Template — Quick Reference

**Location:** `~/projects/basaltor`

## Foundation status
- Better Auth + Drizzle + PostgreSQL wired in
- Mailpit docker service for local verification emails
- Protected routes: `/dashboard`, `/profile`, `/admin`
- RBAC helper layer with sharp shadcn theme consistency
- Automated checks: lint, typecheck, vitest, build

## Local dev
```bash
bun run db:up
bun run db:migrate
bun run seed:dev
bun run dev
```

## Test credentials
- Generated by `bun run seed:dev`
- Printed only in the terminal, not embedded in the frontend
